# MoneyMap – Personal Finance Budgeting App

A web application built as part of a team project to help users track expenses and savings.

## Features
- Track income and expenses  
- View spending patterns  
- Simple and user-friendly interface  

## Tech Stack
- Frontend: React, TypeScript  
- Backend: Node.js, Express  
- CI/CD: GitLab CI
